## Visual QA

- [ ] Title matches the represented data.
- [ ] Axes, units, period, legend, and denominator are legible.
- [ ] Derived-text description agrees with the image.
- [ ] Caption and nearby narrative were reviewed.
- [ ] Source and methodology are identified.
- [ ] The evidence is current enough for the claim.
- [ ] The visual does not imply causation or precision absent from the source.
- [ ] Access and redistribution restrictions are respected.

## Exhibit note

For each selected visual provide:

1. one-sentence takeaway;
2. why it matters to the decision;
3. what it does not establish;
4. source label, date, locator, and evidence ID;
5. recommended memo or deck section.
